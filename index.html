프로그램 소개

이 웹 기반 음성인식 프로그램은 마이크를 통해 사용자의 말을 텍스트로 변환해주는 간단하고 편리한 도구입니다. 브라우저만 있으면 설치 없이 바로 사용할 수 있습니다.

사용 방법

  1. 브라우저에서 프로그램 사이트를 엽니다.
  2. “음성인식 시작” 버튼을 클릭합니다.
  3. 마이크 사용 권한을 허용합니다.
  4. 음성을 말하면 자동으로 텍스트로 변환됩니다.
  5. 완료 후 텍스트를 복사하거나 저장할 수 있습니다.
