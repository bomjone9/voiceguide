프로그램 소개

이 웹 기반 음성인식 프로그램은 마이크를 통해 사용자가 말하면 자동으로 작동해주는 간단하고 편리한 도구입니다. 브라우저만 있으면 설치 없이 바로 사용할 수 있습니다.

사용 방법

  1. 버튼을 눌러 실행, '종료'를 말하여 종료.
  2. 마이크 사용 권한을 허용합니다.
  3. 음성을 말하면 프로그램이 인식합니다.
  4. 사이트+검색어를 말하면 해당 사이트에서 검색됩니다.
  5. 계산기+사칙연산을 말하면 계산 결과를 제공합니다.
  6. 뒤로가기, 우클릭, 좌클릭 음성 명령 실행 가능.
  7. 이용 가능 사이트: 유튜브, 구글, 네이버, 계산기, 메모장
